# Enhancing Accessibility with Gesture-Controlled Virtual Input

## Project Overview
This project is an AI/ML-based virtual mouse and keyboard system that allows users to control computer functions using hand gestures. The system improves accessibility and enables touchless interaction between humans and computers.

The project uses real-time hand tracking and gesture recognition to perform mouse movements, clicks, scrolling, and keyboard operations.

---

## Features
- Virtual mouse control using hand gestures
- Gesture-based keyboard interaction
- Real-time hand tracking
- Touchless computer interaction
- User-friendly and interactive system
- Accessibility support for users

---

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Computer Vision

---

## Project Files
- `app.py` → Main application file
- `mouse.py` → Virtual mouse functionalities
- `keyboard.py` → Virtual keyboard functionalities
- `HandTrackingModule.py` → Hand detection and tracking module
- `index.html` → Basic frontend interface

---

## How It Works
1. The webcam captures hand movements in real time.
2. MediaPipe detects hand landmarks.
3. Different gestures are recognized.
4. Gestures are mapped to mouse and keyboard actions.
5. The system performs the required operation on the computer.
